# Wenzhi Lyu - Personal Academic Website

This is the repository for my personal academic website, hosted using GitHub Pages.

## Website Structure

The website consists of the following pages:
- **Bio**: Brief introduction and overview of my academic background
- **CV**: Comprehensive curriculum vitae with download option
- **Research**: Research interests, publications, and projects
- **Teaching**: Teaching experience and philosophy

## Development

This website is built with HTML, CSS, and TypeScript, with Vite as the build tool.

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/YourUsername/YourUsername.github.io.git
cd YourUsername.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build the site for production:
```bash
npm run build
```

5. Deploy to GitHub Pages:
```bash
npm run deploy
```

## Customization

To update the website content:
- Edit the HTML files for each page
- Modify the CSS files in the `src/styles` directory
- Update your CV PDF in the `src/assets` directory
